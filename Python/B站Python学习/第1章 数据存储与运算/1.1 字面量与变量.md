# 第1章 数据存储与运算
## 1.1、字面量与变量
1. 字面量：直接书写的固定值（数据）
```python
# 字面量的写法  
print(100)  # 整数（int）  
print(3.14)  # 浮点数/小数（float）  
print(True)  # 布尔(bool)  
print(False)  # 布尔(bool)  
print("hello python")  # 字符串(str)  
print("-------")  # 字符串(str)  
print(None)  # 空值(NoneType)

# 布尔类型的本质也是整数类型，其中True为1，False为0  
print(True+1) # 输出结果为2  
print(False-1) # 输出结果为-1
```

 2. 变量
 + 变量：程序中用来存储**单个数据**的**容器**(不是指数据)，通常会把*发生变化的数据*存储在变量中。
 + 定义格式：变量名=变量的值
                        例如：num=3.1415。此时容器的名字就是num，=是把右侧的值赋给左侧，3.1415是变量的值
 ```python
 # 变量  ---> Python是动态类型语言，一个变量可以存储不同类型的数据（但是项目开发中，推荐变量只存储一种类型的数据）
num = 1114.1  
print(num)  # 输出值为1114.1
  
num = num + 1  
print(num)  # 输出值为1115.1

# 练习  
# 基础播放量为20.7万，每月新增播放量为50万，请输出未来来两个月每个月的播放总量？  
bace = 20.7  # 基础播放量  
incr = 50  # 每月新增播放量  
print("第一个月的播放总量：", bace + incr)  
print("第二个月的播放总量：", bace + incr * 2)  

# 升级：一次性定义多个变量  
bace, incr = 20.7, 50  
print("第一个月的播放总量：", bace + incr)  
print("第二个月的播放总量：", bace + incr * 2)
 ```
