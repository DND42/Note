# 第1章 数据存储与运算
## 1.3、字符串
1. 字符串定义的3种方式：
```python
# 双引号定义
s1 = "Hello"

# 单引号定义
s2 = 'Python'

# 三个双引号定义（多行字符串）
s3 = """
	尊敬的王者荣耀用户，
	三角洲不好玩。
"""
```
2. 如果要定义英文的话，常用到转义字符。
![[Pasted image 20260629225316.png|485]]
```python
# 转义字符 \' \" \n \t
msg1 = 'It\'s very good' # 单引号
mag2 = "Hello的意思是\"您好\"" # 双引号
mag3 = "尊敬的王者荣耀用户，\n三角洲不好玩。" # 换行
mag4 = "\t尊敬的王者荣耀用户，\t\n三角洲不好玩。" # 增加缩进
```
