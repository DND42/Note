## 1.2、常见数据类型
Python中常见的***基础***数据类型：
                                                int(整数)，数字类型，如10   -5
                                                float(浮点数)，数字类型，如8.5    3.14    1.0    -3.5
                                                str(字符串)，用引号引起来的都是字符串，如"Python"
                                                bool(布尔)，布尔类型，描述真和假，如True    False
                                                NoneType(空值)，表示**空**或**无值**，仅包含一个值None
![](屏幕截图%202026-06-22%20231713.png)

1. 通过type()语句可以查看数据的类型：type(要查看类型的数据)
                                                                       **如果type里面为变量，则输出的是变量的值的类型
```python
#常见的数据类型 ---> type() 获取指定的字面量或变量的类型
print("Hello World!")  
print(type("Hello World!"))  # 输出结果为<class 'str'>，其中class为类型的意思
print(type(3)) # int  
print(type(3.14)) # float  
print(type(None)) # NoneType  
print(type(True)) # bool  
print(type(False)) # bool
```

2. 通过isinstance()可以检查数据是否属于指定的类型，返回的是一个bool值：isinstance(数据，类型)
```python
# 常见数据类型 ---> isinstance(数据，类型) --> 判定数据是否是指定的类型，如果是：True，如果不是：False  
print(isinstance(3.14,float)) # 输出结果为True
```
